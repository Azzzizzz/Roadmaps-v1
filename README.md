# Engineering Interview Roadmaps

Foundations → advanced learning roadmaps in Markdown, structured so **fresher, mid, and
senior** interviews can be cleared with strong fundamentals — through **exercise- and
project-based learning**.

> These are **skeletons/curricula**, not tutorials. Each file gives you *what to learn, what
> to build, and what to be ready for* — the learning you do yourself. See
> [`CLAUDE.md`](CLAUDE.md) for the rules and [`templates/ROADMAP-TEMPLATE.md`](templates/ROADMAP-TEMPLATE.md)
> for the structure every roadmap follows.

## How to use (cohort)

1. Pick your level: 🟢 Fresher · 🟡 Mid · 🔴 Senior. Each roadmap has all three, segmented.
2. Work modules top to bottom. Do the **exercises**; ship the **mini-projects** and **capstone**.
3. Track progress with the checkboxes. Use the **Interview Question Bank** to drill before rounds.

## Levels

| | Level | Bar |
|---|---|---|
| 🟢 | **Fresher (0–1 yr)** | Fundamentals; can implement basics; clears screening rounds |
| 🟡 | **Mid (2–5 yr)** | Applies concepts in real systems; reasons about trade-offs |
| 🔴 | **Senior (5+ yr)** | Designs at scale; deep trade-offs; mentors; drives architecture |
